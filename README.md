# 🔐 API Security Risk Analysis

A hands-on API security assessment conducted on the JSONPlaceholder REST API as part of the Future Interns Cyber Security Internship(Task 3).

## 🚀 Project Overview

This project focuses on analyzing REST API endpoints to identify common security risks such as unauthorized access, excessive data exposure, resource enumeration, and improper error handling.

The assessment was performed using Postman by testing multiple API endpoints and reviewing responses against API security best practices.

## 🛠️ Tools & Technologies
* Postman
* JSONPlaceholder API
* GitHub
* Canva

## 🔎 Security Assessment Highlights

✅ Tested multiple API endpoints

✅ Analyzed authentication and access control

✅ Evaluated exposed user data

✅ Assessed resource enumeration risks

✅ Reviewed API error handling behavior

✅ Created risk matrix and security dashboard
